# Peminjaman-Buku
## Clone WEB
```
git clone https://github.com/linlaz/Peminjaman-Buku.git
```
```
cd Peminjaman-Buku
```
```
copy .env.example .env
```
```
composer install
```
```
php artisan storage:link
```
```
php artisan key:generate
```
```
php artisan migrate
```
```
php artisan db:seed
```
```
php artisan serve
```
